## ✦ What is CareShift?

CareShift is a **cinematic, immersive health awareness web experience** designed to help people overcome the emotional and psychological barriers that stop them from seeking medical care.

Rather than lecturing or listing symptoms, CareShift takes users on a guided 8-step **interactive journey** — shifting their perspective on healthcare from fear-driven avoidance to informed, empowered action. Everything is anonymous. Nothing is stored.

## ✦ The Journey — 8 Immersive Slides

CareShift's core experience is split into 8 sequential, interactive sections:

| # | Section | Description |
|---|---------|-------------|
| 1 | **Barriers** | Users select what's holding them back today (fear, cost, time, stigma, etc.), and receive a personalised pathway forward |
| 2 | **Real Stories** | Anonymous testimonials from people who were in the same place — and took the step |
| 3 | **CareBot** | An AI-style conversational chat assistant providing confidential, empathetic guidance |
| 4 | **Reality Check** | A 5-question micro-assessment that validates the user's concern with a visual score meter |
| 5 | **Cost of Waiting** | An interactive slider that visualises how early detection saves thousands vs. late-stage treatment costs |
| 6 | **Specialist Finder** | Symptom chips that map to the right type of specialist, with PM-JAY cost estimates |
| 7 | **Community Wall** | An anonymous masonry wall where users can post and read real confessions |
| 8 | **Resources & Helplines** | Verified Indian national helplines for mental health, emergency care, and health coverage |

## ✦ Features

- **Cinematic UI** — Noise-texture SVG overlay, ambient orbs, GSAP scroll animations, and Three.js 3D canvas background
- **Custom Cursor** — Smooth dot + ring cursor with interactive states
- **Dark / Light Mode** — Theme persists across page transitions via Context API
- **Fully Anonymous** — No data is stored, no login required
- **India-First Resources** — Helplines include Tele-MANAS (14416), KIRAN (1800-599-0019), PM-JAY (14555), National Health Line (104), and Emergency (108)
- **Responsive Design** — Adapts to mobile and desktop viewports
- **Smooth Page Transitions** — Cinematic entrance/exit overlays between landing and journey powered by React Router

## ✦ Getting Started

Clone the repository and run the development server:

```bash
git clone https://github.com/swastiksinha1/CareShift.git
cd CareShift
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
